# 🎬 Movie Recommendation System

A content-based movie recommendation system built using Python, Natural Language Processing (NLP), and machine learning techniques.

---

## 🚀 Features

* Recommends movies based on similarity
* Uses movie metadata like genres, keywords, cast, and director
* NLP-based text processing
* Clean and efficient implementation

---

## 🧠 How It Works

1. Data preprocessing and cleaning
2. Feature engineering (combining genres, keywords, cast, etc.)
3. Text vectorization using CountVectorizer
4. Similarity calculation using cosine similarity
5. Recommends top similar movies

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* NLTK

---

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/charankotta32-star/movie-recommendation-system.git
   ```

2. Install required libraries:

   ```
   pip install pandas scikit-learn nltk
   ```

3. Run the script:

   ```
   python main.py
   ```
